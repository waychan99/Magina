# SPButton
## 安装
```
target 'MyApp' do
  pod 'SPButton', '~> 3.0.2'
end
```

## 如何使用
* 创建(创建的时候就设定图片的位置，也可以之后通过属性imagePosition设置)
```
SPButton *button = [[SPButton alloc] initWithImagePosition:SPButtonImagePositionLeft];
```

* 设置图片位置
```
button.imagePosition = SPButtonImagePositionTop;
```

* 设置图文间距
```
button.imageTitleSpace = 10;
```
